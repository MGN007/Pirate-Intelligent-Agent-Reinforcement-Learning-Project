# Pirate-Intelligent-Agent-Reinforcement-Learning-Project
Current &amp; Emerging Trends in AI

Project Overview

For this project, I built a reinforcement learning agent that learns how to navigate a maze to find treasure while avoiding blocked spaces. Some of the environment setup and starter code were provided, including parts of the neural network structure and the maze layout. My role was to complete and refine the deep Q-learning implementation, including building and training the model, adjusting hyperparameters, and debugging issues in the training loop.

I worked a lot on improving the training process. At different points, the agent’s win rate would collapse or training would take too long, so I had to adjust things like exploration rate, reward structure, memory size, and target network updates. Through trial, testing, and careful adjustments, I was able to get the agent to train more consistently and reach a stable solution.

What This Project Taught Me About Computer Science

This course helped me better understand what computer scientists actually do. It’s not just about writing code, it’s about designing systems that solve problems in logical and efficient ways. In this project, I had to think about how to represent the environment as states, what actions the agent could take, and how to define rewards so the model would learn the correct behavior.

I’ve learned to approach problems more systematically. Instead of guessing when something breaks, I now try to analyze the structure of the model, the objective function, and the training data. Reinforcement learning especially showed me that small design choices can have a big impact on outcomes.

Ethical Responsibilities

One big takeaway from this course is that AI systems optimize whatever goal they are given. If the reward structure or objective is poorly designed, the system can behave in unintended ways. That made me realize how important it is to think carefully about how systems are built.

As computer scientists, we have a responsibility to design systems that are reliable, transparent, and aligned with user needs. We also need to communicate limitations clearly and understand that AI models are tools, not perfect decision-makers.
